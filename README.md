# segmented-control-rn

React Native Segmented control

## Installation

```sh
npm install segmented-control-rn
```

## Usage

```js
import SegmentedControlRn from "segmented-control-rn";

// ...

const result = await SegmentedControlRn.multiply(3, 7);
```
